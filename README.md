# DSA-Problems
 
A personal collection of Data Structures & Algorithms problems solved in Java, each stored in its own folder. This repository is automatically managed and kept up to date by the [LeetPush](https://github.com/LeetPushExtension/LeetPush) browser extension, which pushes accepted LeetCode submissions straight to GitHub.
 
## About
 
This repo serves as a log of LeetCode problem-solving progress — a place to track solutions, revisit past approaches, and build a searchable archive of solved problems over time.
 
## Structure
 
Each problem lives in its own folder, named using the pattern:
 
```
<problem-number>-<problem-slug>/
```
 
For example:
 
```
DSA-Problems/
├── 15-3sum/
├── 27-remove-element/
├── 713-subarray-product-less-than-k/
└── README.md
```
 
Inside each folder you'll typically find the Java solution file for that specific problem.
 
## Language
 
- **Java** — all solutions are currently written in Java.
## How It Works
 
1. Solve a problem on LeetCode.
2. The [LeetPush](https://github.com/LeetPushExtension/LeetPush) extension detects the accepted submission.
3. It automatically creates/updates the corresponding folder and pushes the solution to this repository.
## Topics Covered
 
As the collection grows, it spans common DSA patterns and topics such as:

- Sorting & Searching
*(This list will expand as more problems are added.)*
 
## Usage
 
Feel free to browse any folder to see the problem's solution. Each solution is self-contained and can be run independently with a Java compiler:
 
```bash
javac Solution.java
java Solution
```
